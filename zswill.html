WHAT IT IS

System cleaner

ZSWILL is a CLI utility for cleaning Windows that does exactly what it promises. No background processes. No ads. No "optimizers" that slow your computer down further. One file. Open source. Minimal dependencies.

⏎ Safety at code level

Every path that can be cleaned is in a strict allowlist. System32, Program Files, Windows — blocked at the code level. You physically cannot delete system files through ZSWILL.

No "are you sure?" prompts — just a block. If a path is not in the allowlist, it won't be touched. Period.

⏎ Browser detection

ZSWILL checks which browsers are actually installed — Chrome, Firefox, Edge, Opera, Brave. And only cleans those. No "path not found" in logs. Firefox parses profiles via Profiles/*.default.

⏎ Uninstall — completely

Built-in uninstaller reads Windows registry. Launches the native uninstaller. Then hunts for leftovers in Program Files, AppData, ProgramData. Cleans registry keys recursively — even nested ones. No traces left.

DOCS

How it works

ZSWILL is a single exe file. No installation, no dependencies. Download, run, done.

⏎ Launch

Download zswill.exe from releases. Double-click to run, or launch from terminal. No pip install, no Python — everything is packed into one executable.

⏎ Commands

After launch — a menu with 36 cleaners. Keys: 1-36 for individual cleaners, A — run all, S — scan (shows how much can be freed without deleting), U — uninstall programs, C — settings (language, auto-start), 0 — exit.

⏎ Safety

All allowed paths are in SAFE_PATHS in the source code. System32, Program Files, Windows — blocked at the code level. If a path is not in the allowlist, ZSWILL won't touch it. No "are you sure?" — just a block.

⏎ Config

Settings are saved to %APPDATA%\ZSWILL\config.json — stores the selected language. Auto-start via registry HKCU\...\Run. Config is created automatically on first launch.

⏎ Internet Clean

Resets DNS cache (ipconfig /flushdns), Winsock (netsh winsock reset), and TCP/IP (netsh int ip reset). Three commands to fix network issues. Reboot recommended.

⏎ App Cache

Runs WSReset.exe to clear Microsoft Store cache, then scans all packages in %LOCALAPPDATA%\Packages for INetCache, LocalCache, TempState and LocalState\Cache directories. Cleans everything safely.

⏎ Package Cache

Cleans package manager cache: pip cache purge (Python) and npm cache clean --force (Node.js). Falls back to deleting cache folders directly if commands are unavailable.

BROWSERS

Detect by presence

No guessing. ZSWILL checks user data folders — if a browser is not installed, its items simply don't appear.

Chrome

Cache, code cache, Service Workers, history, cookies. Everything Chromium accumulated.

Firefox

FAQ

Frequently asked

Is it safe?
Yes. All paths are in SAFE_PATHS — an allowlist. System32, Program Files, Windows, ProgramData — blocked in code. You cannot delete system files even if you try.
Need admin rights?
Most functions work without admin. DNS cache and Windows Logs work better as admin, but the program never crashes — it just reports nothing to clean.
What about Firefox?
ZSWILL finds Firefox profiles in %APPDATA%\Mozilla\Firefox\Profiles, looks for *.default folders and cleans cache2, thumbnails, places.sqlite, cookies.sqlite.
Can I undo a clean?
No. Deleted files cannot be recovered. Always run scan (S) first to see what will be deleted before cleaning.
How is it different from CCleaner?
Size: 1 file (vs 150+ MB). Price: free, open source. Behavior: no background process, no telemetry, no ads. Animations: yes, and they're beautiful.
How does the uninstaller work?
Reads registry (HKLM + HKCU). Launches native uninstall silently (/S /silent). Searches for leftover folders by program name. Recursively cleans registry keys — not just empty ones, but nested ones too.
How to switch language?
In the app: Settings (C) → Language (1). Switches between Russian/English on the fly. Language is saved to config.json. On the site: RU/EN button in the top-right corner.
What does Windows Updates clean?
%WINDIR%\SoftwareDistribution\Download — the folder with downloaded updates. After updates are installed, these files are no longer needed. Windows will re-download them if needed.
Will it make my computer faster?
Cleaning frees up disk space. If your SSD is full — yes, it helps. But ZSWILL doesn't do "magic registry optimization" or other pseudoscience. Only real cleaning.
Why ZSWILL?
Because existing tools are either paid, send telemetry, weigh gigabytes, or pretend to "optimize" the registry. ZSWILL is clean, transparent, minimal. Just like its code.
What is Internet Clean?
Flushes DNS cache (ipconfig /flushdns), resets Winsock (netsh winsock reset) and TCP/IP stack (netsh int ip reset). Helps with network issues after malware removal or ISP change. Reboot required.
What is App Cache Clean?
Cleans cache for Microsoft Store (WSReset.exe), Xbox Game Pass and all Windows store apps in %LOCALAPPDATA%\\Packages — INetCache, LocalCache, TempState. Also scans every app in %APPDATA% and %LOCALAPPDATA% for Cache/cache/Caches folders. Browser folders are excluded. Safe — each cache is capped at 1 GB to prevent data loss.
What is Package Cache?
Cleans package manager cache: pip (Python) — pip cache purge, and npm (Node.js) — npm cache clean --force. Falls back to deleting cache folders directly if commands are unavailable.
Why trust ZSWILL?
Open source (MIT) — every line of code is on GitHub. Single file — no installer, no hidden modules. No telemetry, no network requests, no background processes. Python — can't execute raw system calls without permission. SAFE_PATHS allowlist — even if you try, system files won't be deleted. You can read the entire program in 10 minutes.
How does messenger detection work?
ZSWILL checks for installed messenger folders in %APPDATA% — WhatsApp, Viber, Telegram Desktop (tdata folder), Skype, Skype for Desktop. Only detected messengers are cleaned. No dead paths.
What AI tools does ZSWILL clean?
Claude, GitHub Copilot, Cline (VS Code extension), DeepSeek, ChatGPT, Codex CLI, Qwen, Gemini, OpenCode, Zed Editor, Mistral/Le Chat, and Ollama models. Each tool's cache directory is checked for presence before cleaning.

ABOUT

Team ZSWILL

ZSWILL — open source system cleaner

We are a team of developers who are tired of "optimizers" that live in the tray, eat memory, and display ads. ZSWILL was born as an alternative: one file, no dependencies besides Python, full transparency.

Every line of code is open. You can read, modify, compile it. No telemetry. No "performance improvement" via registry tweaking. Only deletion of what can actually be deleted.

Built by developers, for users. No investors, no sales team — just code and coffee.
