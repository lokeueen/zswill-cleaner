# 🧹 zswill-cleaner

> ⚠️ **The website is a mess, I know.** I tried my best, but let's be real — it's not great. Just ignore the "download" page, it doesn't matter. The real deal is right here. 😅

**zswill-cleaner** is a lightweight Windows utility that helps you clean up temporary files and system junk to free up disk space. No bloat, no spyware — just a simple `.exe` that gets the job done. 🚀

## ✨ Features

- 🗑️ **Temporary files cleanup** — wipes out all those useless temp files
- 🧽 **System junk removal** — gets rid of cache and unnecessary system leftovers
- 💾 **Free up disk space** — reclaim gigabytes of storage
- ⚡ **Fast & lightweight** — runs in seconds, tiny footprint

## 🛠️ Usage

1. **Download** `zswill-cleaner.exe` (it's the only file in this repo, hard to miss)
2. **Right-click** → **Run as Administrator** (yes, it needs admin rights)
3. **Let it do its thing** — sit back and watch the magic happen 🪄

```
zswill-cleaner.exe
```

> 💡 Pro tip: run it once a month to keep your system fresh.

## 📋 Requirements

- 💻 Windows 7 or later (works best on Windows 10/11)
- 🔑 Administrator privileges (required for cleaning system locations)
- 🧠 A desire for a cleaner PC

## ❓ Why does this exist?

Because Windows likes to hoard junk like a digital hoarder. Temporary files, caches, logs — after a few months it adds up. This tool cleans all that nonsense so you don't have to hunt through `%temp%`, `prefetch`, and other Windows dungeons manually. 🗺️
